# 用户指南

## 快速开始

### 安装
1. 下载并安装Python 3.8或更高版本
2. 克隆或下载项目代码
3. 安装依赖包：
   ```bash
   pip install -r requirements.txt
   ```

### 配置API密钥
1. 运行应用程序
2. 点击左下角的设置按钮
3. 在API设置中输入您的DeepSeek API密钥
4. 保存设置并重启应用

### 启动应用
```bash
python main.py
```

## 功能介绍

### 聊天功能
- 点击左侧"+ 新聊天"按钮创建新的聊天会话
- 在底部输入框输入问题并按回车发送
- AI助手会自动回复，并在需要时执行PowerShell命令

### PowerShell命令执行
当您询问与电脑操作、文件管理、系统信息、网络配置等相关的问题时，AI助手会自动生成并执行相应的PowerShell命令。

示例问题：
- "列出当前运行的进程"
- "查看C盘的剩余空间"
- "显示网络配置信息"

### 设置功能
- 点击左下角的齿轮图标打开设置面板
- 可以配置API密钥、主题、网络代理等选项
- 修改设置后需要重启应用才能生效

## 使用技巧

### 提高回答准确性
- 尽量提供详细和具体的问题描述
- 当需要执行系统操作时，明确说明您的意图
- 对于复杂任务，可以分步骤询问

### 安全注意事项
- 应用会执行AI生成的PowerShell命令，请确保您信任AI的回答
- 不要在涉及敏感数据的场景中使用此应用
- 定期检查执行的命令以确保安全性

## 常见问题

### 启动时报错
- 确保已正确安装所有依赖包
- 检查Python版本是否符合要求
- 确认API密钥已正确配置

### API调用失败
- 检查网络连接
- 确认API密钥是否有效
- 查看是否有防火墙或代理设置问题

### PowerShell命令未执行
- 检查命令格式是否正确
- 确认应用是否有足够的权限执行命令
- 查看命令是否包含危险操作被系统阻止

## 故障排除

### 日志查看
应用日志位于 `logs/` 目录下，可以查看详细的错误信息。

### 重置设置
如果遇到配置问题，可以删除 `config/settings.json` 文件来重置所有设置。
