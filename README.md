# Savvy
这个项目可以通过自然语言来操控电脑完成各种任务，原理是通过大语言模型来将自然语言转化为电脑可操作的命令或者代码

## 功能特点
- 自然语言交互：通过聊天方式与AI助手交流
- PowerShell命令执行：AI可以自动生成并执行PowerShell命令
- 模块化设计：易于扩展和维护的代码结构
- 设置管理：支持多种配置选项，包括API密钥、主题、网络等

## 快速开始

### 安装依赖
```bash
pip install -r requirements.txt
```

### 配置API密钥
1. 运行应用程序
2. 点击左下角的设置按钮
3. 在API设置中输入您的DeepSeek API密钥
4. 保存设置并重启应用

### 启动应用
```bash
python main.py
```

## 系统要求
- Python 3.8 或更高版本
- Windows 10/11 (当前版本主要支持Windows系统)

## 使用示例
1. 询问系统信息："查看我的电脑配置"
2. 文件管理："列出下载文件夹中的所有PDF文件"
3. 网络操作："测试到百度的网络连接"
4. 进程管理："结束所有记事本进程"

## 安全说明
应用会执行AI生成的PowerShell命令，请在使用时确保：
- 了解AI可能执行的操作
- 不在涉及敏感数据的场景中使用
- 定期检查执行的命令
